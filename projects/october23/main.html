Why is the viewport meta tag important?

  • The viewport meta tag is crucial for responsive web design. It controls how a webpage is displayed on different devices, particularly mobile ones.

How can you adjust the CSS for printing?

  • @media print { body { font-size: 12pt; } .no-print { display: none; /* Hides elements that should not be printed */ } /* Additional print styles here */ }

When would you want to adjust the CSS for printing?

  • Business documents: When providing reports, invoices, or proposals that need a professional appearance.
  • User-generated content: For user-generated pages (like articles or recipes) where a clean print format enhances usability.
  • Long-form content: To ensure that articles or blog posts are readable and well-structured when printed.
  • Avoiding distractions: To hide elements that aren't useful on paper, like navigation bars, ads, or other interactive elements.
