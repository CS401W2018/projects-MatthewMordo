Engagement: DNS and SSL

What is DNS?

  • DNS (Domain Name System) is like the phonebook of the internet. It translates human-readable domain names (e.g., www.example.com) into IP addresses (e.g., 192.168.1.1) that computers use to identify each other on the network. Without DNS, you'd need to remember long strings of numbers instead of easy-to-remember domain names.

How do you get a domain name?

  • Firstly, come up with a domain name, then using domain name registrars such as GoDaddy to see if the domain name you want is available.
