Internet

The internet is a global network of interconnected devices and computer networks that allows users to access websites, information, or communicated with an another person.

Web

The web or WWW(World Wide Web) is a system of inter-connected web pages or websites that are accessible on the internet.

How are the Internet and Web different?

The difference of the internet and the web is that the internet is the underlying infrastructure that connects computers and other devices globally, while the web is a service that operates on top of the internet, allowing access of to utilities and share information through web pages such as facebook,wikipedia,etc...

ASCII and Unicode

ASCII(American Standard Code for Information Interchange) is a character encoding standard for electronic communication, representing text in computers. Unicode on the other hand is a more extensive character encoding standard that supports characters from virtually all languages and symbols worldwide.

What is UTF-8?

UTF-8 is a variable-width character used for electronic communication. It can encode all Unicode characters and is backward-compatible with ASCII.

URL

URL or (Uniform Resource Locator) is the address used to access resources on the web, such as webpages, images, videos, etc...

Domain Name

A domain name is an adress that indentifies websites on the internet. An example of a domain name is "Google.com"

HTTP

HTTP or (Hypertext Transfer Protocol) is a protocol used to transfer data over the web. This enables communication between a web server and a web browser.

HTTPS

HTTPS or (Hypertext Transfer Protocol Secure) is a secure version of HTTP. This uses encryption to ensure the safe transfer of data between a web server and a web browser.

IP Address

An IP Address or (Internet Protocol Address) is a unique numeric identifier assigned to each device connected to the internet. This allows the device to connect and communicate with other devices locally or wirelessly.

DNS

A DNS or (Domain Name System) is a system that translates domain names into IP addresses. This allows users to access websites using human-readable names instead of numerical addresses.

W3C Recommendations

W3C Recommendations are guidelines and practices developed by the World Wide Web Consortium (W3C) in promoting consistency and best practices in web development

XML, XHTML, HTML, HTML5

XML (eXtensible Markup Language) is a markup language used for defining data structures.

XHTML (eXtensible Hypertext Markup Language) is a stricter version of HTML that follows XML rules.

HTML (Hypertext Markup Language) is the standard markup language for creating web pages.

HTML5 is the latest version of HTML, adding new features and capabilities, such as multimedia elements and enhanced support for mobile devices.
