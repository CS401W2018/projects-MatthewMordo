  2. .. Parent Folder
  3. /at beginning/ start at the root

Do links need to be inside the NAV tag?

NOT all links of a document should be inside a nav element. The nav element is intended only for major blocks of navigation links.

Explain the difference between an absolute link and a relative link. Why would you use relative links?

An absolute link (or absolute URL) provides the full web address needed to access a resource, regardless of where the link is being referenced from.

A relative link gives the location of a resource relative to the current page or the site’s root. It doesn’t include the full web address, so the link is determined based on the page from which it's called.

Why relative links?

Relative links are easier to manage within a site because they continue to work if the domain changes, as long as the internal structure stays the same. They're also more portable, making it easy to move a site to different servers without updating the links. Additionally, relative links are shorter and quicker to type or update compared to absolute links.
