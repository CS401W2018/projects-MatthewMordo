What is the difference between a DIV tag and HEADER, MAIN, FOOTER?

DIV Tag:

A generic container for flow content, with no semantic meaning.

HEADER:

Semantic element that represents introductory content or navigational links for a section or page.

MAIN:

Represents the dominant content of the "body" of a document. There should only be one "main" per page, focusing on the primary content.

FOOTER:

Semantic element that represents the footer of a section or page, typically containing metadata, copyright information, or related links.

What are different ways to select a color?

Named Colors:

Use predefined color names like red, blue, etc.

Hexadecimal Colors:

Specify colors using a six-digit hex code.

RGB Colors:

Define colors using the RGB color model.

RGBA Colors:

Similar to RGB but includes an alpha channel for transparency.

HSL Colors:

Define colors using hue, saturation, and lightness.

HSLA Colors:

Similar to HSL but includes an alpha channel for transparency.
