Why do we need a viewport meta tag?

  • The viewport meta tag is essential for responsive web design. It allows you to control the layout of a webpage on mobile browsers. By setting the viewport to match the device's width and scaling, you ensure that your website displays correctly across different screen sizes.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

  • When you have the same property set twice, the one that appears last in your CSS will take priority due to the CSS cascade.

How do you change the CSS for a web page displayed on a phone?

  • To change the CSS for a webpage displayed on a phone, you typically use media queries. Media queries allow you to apply different styles based on the device's characteristics, like width.
