What is the difference between a block and an inline tag? Give examples of each one.

  • Block Tags: elements take up the full width available and start on a new line. Examples include div, h1, p, and section.
  • Inline Tags: These elements only take up as much width as necessary and do not start on a new line. Examples include span, a, img, and strong.

What is normal flow in HTML?

  • Normal flow refers to the default layout of elements in a web page, where block-level elements stack vertically and inline elements sit next to each other horizontally. In normal flow, the browser lays out elements in the order they appear in the HTML document.

What does float do? Why do you need to clear the float?

  • Float: The float property allows elements to be taken out of the normal flow and positioned to the left or right of their container. Text and inline elements will then wrap around the floated element.

What do absolute, fixed, and relative do?

  • Absolute Positioning: Positions an element relative to its nearest positioned ancestor (an ancestor with a position other than static). If no such ancestor exists, it uses the document body.
  • Fixed Positioning: Positions an element relative to the viewport, meaning it stays in the same place even when the page is scrolled.
  • Relative Positioning: Positions an element relative to its normal position in the flow. It allows for slight adjustments using top, right, bottom, and left properties without affecting the layout of surrounding elements.

How can you make nice navigation buttons for mobile devices?

  • Use a responsive design framework like Bootstrap or Flexbox for flexibility.
  • Ensure buttons are large enough for easy tapping (at least 44x44 pixels).
  • Use clear, contrasting colors and readable fonts.
  • Add hover and active states for interactivity.
  • Consider using icons alongside text for better visual appeal.
