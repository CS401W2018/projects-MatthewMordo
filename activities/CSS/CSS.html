Explain the difference between inline, embedded, and external CSS.

Inline CSS is directly in an HTML file using the "Style" Attribute.

Embedded CSS is inside a "style" and "head" section of an HTML document.

External CSS is in a separate ".css" file linked via a "link" tag in the "head"

Explain why you should only use external CSS.

Clarity: Keeping HTML and CSS separate makes the code easier to read and maintain. HTML handles structure, while CSS handles presentation.

Multiple Pages: You can link the same CSS file to multiple HTML documents, ensuring consistent styling across an entire website without duplicating code.

What is a selector used for?

A selector in CSS is used to target HTML elements that you want to style. It defines which elements the styles will apply to. Here are the main purposes and types of selectors:

Purpose of Selectors:

Targeting Elements: specify which HTML elements to style based on their type, class, ID, attributes, and more.

Applying Styles: an element is targeted, the corresponding CSS rules are applied to modify its appearance (e.g., color, font, layout).
