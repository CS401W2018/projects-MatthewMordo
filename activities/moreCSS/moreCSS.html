    What is the difference between block tags and inline tags.

  • Block Tags: These elements take up the full width available, creating a new line before and after the element. They are typically used for larger content sections.
  • Inline Tags: These elements only take up as much width as necessary and do not create a new line. They are often used for smaller pieces of content within block elements.

    • What is the margin, border, and padding?

  • Margin: The space outside an element's border. It creates space between the element and other elements around it. Margins can be set individually for each side (top, right, bottom, left).
  • Border: A line that goes around an element, defined by its thickness, style, and color. Borders are part of the box model and can be customized in various ways.
  • Padding: The space between an element's content and its border. Padding creates space inside the element itself. Like margins, padding can also be set individually for each side.

    • What are units of measurement you can use in CSS?

    Absolute units:

  • Pixels: A fixed unit, commonly used for precise sizing.
  • Centimeters: Generally used for print styles.
  • Inches: Also used for print.

    • When would you use a figure tag?

    You would use the figure tag when you want to semantically group media content (like images, illustrations, or diagrams) with an optional caption. This is useful for:
  • Improving accessibility by clarifying the relationship between the media and its description.
  • Enhancing the structure of your HTML, making it clear that the content is a standalone reference.
  • Allowing for easier styling and layout adjustments for media content.
